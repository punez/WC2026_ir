# 🏆 راهنمای راه‌اندازی بات جام جهانی ۲۰۲۶

---

## قدم ۱ — ساختن بات در تلگرام

1. در تلگرام **@BotFather** رو پیدا کن
2. بنویس `/newbot`
3. اسم بده → یوزرنیم بده (باید به `bot` ختم بشه)
4. **Token** رو کپی کن — بعداً لازمه

---

## قدم ۲ — گرفتن آیدی عددی ادمین‌ها

1. تلگرام → **@userinfobot** → پیام بده
2. عدد `Id` رو کپی کن
3. برای هر ادمین این کار رو بکن

---

## قدم ۳ — GitHub

1. برو **github.com** → اکانت بساز
2. **New repository** → اسم بده → **Create**
3. روی **"uploading an existing file"** کلیک کن
4. همه فایل‌های زیر رو آپلود کن:

```
bot.py
config.py
database.py
wc_data.py
utils.py
requirements.txt
Procfile
handlers/
    __init__.py
    user.py
    admin.py
```

5. **Commit changes**

---

## قدم ۴ — Supabase (دیتابیس رایگان)

1. برو **supabase.com** → اکانت بساز
2. **New Project** → اسم بده → رمز بذار → **Create**
3. بعد از ساخت برو: **Settings → Database → Connection string → URI**
4. کپی کن — این میشه `DATABASE_URL`

> ⚠️ اگه نخوای Supabase راه بندازی، بات از SQLite محلی استفاده می‌کنه.
> فقط برای Railway باید یه **Volume** اضافه کنی (قدم ۵ رو ببین).

---

## قدم ۵ — Railway (هاست رایگان)

1. برو **railway.app** → با GitHub لاگین کن
2. **New Project → Deploy from GitHub repo**
3. Repo رو انتخاب کن

### تنظیم Variables:
برو به **Variables** و اینا رو اضافه کن:

| Key | Value |
|-----|-------|
| `BOT_TOKEN` | توکن از BotFather |
| `ADMIN_IDS` | مثلاً `123456789,987654321` |
| `DATABASE_URL` | از Supabase (اختیاری) |

### اگه از SQLite استفاده می‌کنی (بدون Supabase):
برو به **Volumes** در Railway → **Add Volume**:
- Mount path: `/data`

این کار باعث میشه دیتابیس بعد از ری‌ستارت پاک نشه.

### Deploy:
روی **Deploy** کلیک کن — تمومه! ✅

## دستورات بات

### کاربر عادی:
| دستور | کار |
|-------|-----|
| `/start` | شروع + انتخاب زبان |
| منوی اصلی | همه کارها با دکمه |

### ادمین:
| دستور | کار |
|-------|-----|
| `/admin` | پنل ادمین |
| `/sendall متن` | پیام همگانی |

### از پنل ادمین:
- **📋 لیست بازی‌ها** — همه بازی‌ها با وضعیتشون
- **✅ ثبت نتیجه** — نتیجه رو ثبت کن، امتیاز خودکار حساب میشه
- **➕ بازی جدید** — بازی‌های حذفی رو اضافه کن
- **📢 پیام همگانی** — به همه کاربرها پیام بده

---

## سیستم امتیازدهی

| حالت | امتیاز |
|------|--------|
| ✅ نتیجه دقیق (مثلاً ۲-۱ زدی، ۲-۱ شد) | +۱۰ |
| 📐 تفاضل درست (۳-۱ زدی، ۲-۰ شد) | +۷ |
| ✔️ فقط برنده درست | +۵ |
| 🎟 شرکت در پیش‌بینی | +۲ |
| 🔥 فینال | همه × ۲ |

**نکته مهم:** امتیاز شرکت (+۲) همیشه داده میشه. بقیه روش اضافه میشن.
مثلاً نتیجه دقیق = ۲ + ۱۰ = ۱۲ امتیاز

## جریان کار

```
کاربر /start میزنه
    ↓
انتخاب زبان (فارسی/انگلیسی)
    ↓
منوی اصلی
    ↓
⚽ پیش‌بینی بازی‌ها → انتخاب مرحله → لیست بازی‌ها → پیش‌بینی
📊 امتیاز من
🏆 جدول امتیازات
```

---

## نکات مهم

- بازی‌ها **خودکار** قفل میشن (هر ۳۰ ثانیه چک میشه)
- کاربرها تا لحظه شروع بازی می‌تونن **تغییر بدن**
- ادمین نتیجه رو ثبت کنه → همه امتیازها **فوری** حساب میشه
- شناسه کاربر: **نام + آیدی عددی** (یونیک و همیشگی)
- بات کاملاً **دوزبانه** — هر کاربر زبان خودشو انتخاب می‌کنه
